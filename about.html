About Investment Backtest Analyzer

The Investment Backtest Analyzer is a comprehensive Python-based tool designed for sophisticated investment strategy analysis and backtesting. Built for investors, analysts, and researchers who need professional-grade performance analysis.

What It Does

This tool allows you to:

  • Follow and analyze the performance of multiple investors simultaneously
  • Backtest investment strategies across different time periods (2012-2025)
  • Compare your strategy against market benchmarks (S&P 500, NASDAQ)
  • Calculate key performance metrics like CAGR, Alpha, and Sharpe ratios
  • Generate detailed reports and visualizations
  • Export transaction logs and performance data

Key Features

Multi-Investor Tracking

Track up to 5 investors simultaneously with different selection criteria based on historical performance, portfolio value, or CAGR metrics.

Flexible Timing

Configure buy/sell quarters, data source periods, and lookback windows to match your investment strategy.

Rebalancing Options

Choose between rebalancing and buy-and-hold strategies to see how different approaches affect returns.

Advanced Filtering

Set minimum portfolio percentage thresholds and apply various selection modes for investor filtering.

Performance Analysis

Comprehensive performance metrics including alpha calculation, CAGR analysis, and benchmark comparisons.

Data Export

Export detailed transaction logs, performance reports, and generate publication-ready charts.

Technical Stack

Built with modern Python technologies:

Python 3.x Pandas Matplotlib NumPy yfinance CSV Data Processing

Configuration Options

The tool offers extensive configuration options that can be adjusted in the main script:

NUM_INVESTORS = 5 # Number of investors to track START_YEAR = 2012 # Backtest start year END_YEAR = 2026 # Backtest end year HAS_RELOCATION = True # Enable/disable rebalancing SELECTION_MODE = 3 # Investor selection method TOP_PICKS_PER_INVESTOR = 3 # Number of stocks per investor MIN_TOP1_PCT = 15 # Minimum portfolio percentage threshold

Selection Modes

Mode 0: Random Selection

Randomly selects investors from available dataset with coverage analysis.

Mode 1: Prior Performance

Selects investors based on their performance in the previous year.

Mode 2: Portfolio Value

Ranks investors by their total portfolio value in the previous quarter.

Mode 3: CAGR 3-Year

Uses 3-year compound annual growth rate for investor ranking.

Mode 4: Cumulative CAGR

Ranks based on cumulative CAGR over all available historical data.

Output and Reports

The analyzer generates several types of output:

  • Transaction Logs: Detailed CSV files with all buy/sell transactions
  • Performance Charts: Visual comparison of your strategy vs benchmarks
  • Annual Performance: Year-by-year breakdown with alpha calculations
  • CAGR Analysis: Compound annual growth rate calculations
  • Track Records: Quarterly and annual investor performance history
  • Batch Analysis: Run multiple scenarios and compare results

Data Requirements

The tool works with CSV files containing:

  • Historical stock prices (daily and quarterly)
  • Investor portfolio positions by quarter
  • Benchmark data (S&P 500, NASDAQ)
  • Portfolio allocation percentages
